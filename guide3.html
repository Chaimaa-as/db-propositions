01 Se connecter à l'administration ›

Avant de pouvoir rédiger quoi que ce soit, il faut accéder au back-office WordPress.

  1. Ouvre Safari ou Chrome et tape : dadais.me/wp-admin — mets-le en favori avec ⌘ D.
  2. Saisis ton identifiant (ou ton adresse e-mail) et ton mot de passe.
  3. Clique sur Se connecter. Tu arrives sur le Tableau de bord.

03 Rédiger dans les deux langues — Polylang ›

Chaque contenu existe en deux versions totalement indépendantes — il n'y a jamais de mélange FR/EN.

Créer la version anglaise d'un contenu

  1. Ouvre ou crée ton article en français.
  2. Dans la colonne de droite, repère le bloc Langues.
  3. À côté du drapeau 🇬🇧, clique sur le petit +.
  4. Une nouvelle page vierge s'ouvre, reliée à la version FR. Rédige ta traduction.
  5. Publie les deux versions séparément.

07 Entretien du site — Mises à jour ›

Des pastilles rouges ou des bandeaux orange apparaissent dans le menu de gauche quand une mise à jour est disponible.

  1. Va dans Extensions › Extensions installées.
  2. Si un bandeau orange indique Une nouvelle version est disponible, clique sur mettre à jour maintenant.
  3. Répète pour chaque extension signalée.
Extensions › Extensions installées
Les bandeaux orange signalent les mises à jour. Ici : Polylang et WP All Import ont chacun une version plus récente.

Mises à jour majeures de WordPress

Pour un saut de version majeure (6.x → 7.x), contacte-moi d'abord. Les mises à jour mineures sont sans risque.

B — Import des images (galeries)

  1. Regroupe toutes tes photos dans un seul dossier sur ton Mac.
  2. Dans ce dossier, crée un fichier texte nommé exactement liste_photos.txt (TextEdit → Format → Texte brut).
  3. Remplis le fichier — une photo par ligne.
nom-du-fichier.jpg : tag1, tag2, tag3

hotel-nice.jpg   : voyage, hotel, cote d'azur
plage-galets.jpg : mer, soleil, baignade
081127-veste.jpg : la belle allure, mode, image
